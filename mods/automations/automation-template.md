# Automation Template

## Structure

- **Name**: Automation name
- **Schedule**: When it runs (daily, weekly, monthly, etc.)
- **Trigger**: What triggers it
- **Steps**: Process steps
- **Notifications**: Who gets notified
- **Error Handling**: How errors are handled

## Creation Steps

1. Create automation file
2. Define schedule and triggers
3. Document process steps
4. Set notifications
5. Add to automation-registry.md
