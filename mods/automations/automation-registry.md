# Automation Registry

## Automations

1. **daily-project-review** - Daily project status review
2. **weekly-roadmap-review** - Weekly roadmap update
3. **data-refresh-check** - Check data freshness
4. **model-retraining-check** - Check model performance
5. **dashboard-quality-check** - Validate dashboard data
6. **broken-link-check** - Find broken documentation links
7. **repo-cleanup-check** - Repository maintenance checks
8. **changelog-generator** - Generate changelog from commits
