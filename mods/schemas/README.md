# Schemas

This directory contains JSON schemas for validating framework artifacts. Use these schemas when creating new agents, skills, prompts, workflows, and other structured files.

## Schemas

| File | Validates |
|------|-----------|
| agent.schema.json | Agent definitions (`AGENT.md` frontmatter) |
| skill.schema.json | Skill definitions (`SKILL.md` frontmatter) |
| prompt.schema.json | Prompt definitions (`.prompt.md` frontmatter) |
| workflow.schema.json | Workflow definitions (`.workflow.md` frontmatter) |
| task.schema.json | Task entries in task files |
| automation.schema.json | Automation definitions |
| eval.schema.json | Evaluation definitions (`.eval.json`) |
| memory.schema.json | Memory log entries |
| project.schema.json | Project type, purpose, stack, data, and constraints |
