# Memory

This directory contains project memory, decisions, and learnings. These files are append-only runtime documents — they **start empty** and accumulate content as agents run and decisions are made. Do not expect pre-filled data on a fresh clone.

## Contents

- **project-memory.md**: Key project information and context that persists across sessions
- **decision-log.md**: Important decisions made, with rationale and date
- **assumptions-log.md**: Key assumptions being made during development
- **lessons-learned.md**: Lessons learned from agent runs and project work
- **reusable-patterns.md**: Reusable patterns identified for future re-use
- **agent-feedback-log.md**: Agent feedback and learnings from runs
