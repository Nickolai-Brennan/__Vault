# Templates

This directory contains reusable templates for creating new content.

## Templates

- agent-template.md
- skill-template.md
- prompt-template.md
- workflow-template.md
- task-template.md
- eval-template.md
- report-template.md
- changelog-template.md
- project-template.md — question model for identifying project type, structure, and purpose

## Project Type Templates

The `project-types/` subdirectory contains one specialized template per project type. Each template extends the general `project-template.md` with sections and questions tailored to that project type. Use the template that matches the primary type selected in `PROJECT.md`.
