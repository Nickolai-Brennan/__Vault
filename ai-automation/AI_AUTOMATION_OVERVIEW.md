# AI Automation Overview

This document provides a high-level overview of the AI automation system.

## Purpose

The AI automation framework enables orchestrated, guided AI agent execution with proper governance, quality assurance, and memory management.

## Key Components

### Agents
- Specialized AI agents for specific tasks
- Agent registry and templates
- Agent-specific prompts, rules, and workflows

### Skills
- Reusable, modular capabilities
- Skill registry with documentation
- Shared across multiple agents

### Workflows
- Multi-step processes
- Agent orchestration
- Error handling and fallbacks

### Governance
- Security policies
- Quality standards
- Access controls

## Directory Structure

- **agents/**: Agent definitions and configurations
- **skills/**: Reusable skill modules
- **prompts/**: Prompt templates and definitions
- **workflows/**: Workflow orchestration files
- **instructions/**: Global instructions and standards
- **automations/**: Automated task definitions
- **tasks/**: Task tracking and management
- **memory/**: Project memory and decision logs
- **evals/**: Evaluation frameworks and results
- **scripts/**: Utility scripts
- **schemas/**: JSON schemas for validation
- **templates/**: Reusable templates
- **references/**: Reference documentation
- **logs/**: Run logs and audit trails
